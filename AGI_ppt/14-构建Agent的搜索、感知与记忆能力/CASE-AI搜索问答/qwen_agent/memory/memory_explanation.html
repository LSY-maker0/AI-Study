Qwen Agent Memory 类详解

概述

Memory 是一个特殊的代理类，专门用于文件管理和知识检索。它实现了RAG（检索增强生成）功能，允许系统在处理用户请求时参考上传的文档或预先配置的知识库文件。

核心功能：将文件内容作为知识库存储，并在用户提问时检索相关内容以增强回答质量。

主要特性

  • 文件管理：自动处理消息中的上传文件，将其纳入知识库范围
  • RAG支持：集成检索增强生成技术，提升回答准确性
  • 多种文件格式支持：包括PDF、Word、PowerPoint、Excel、TXT、CSV等多种常见格式
  • 智能关键词生成：根据用户查询自动生成检索关键词
  • 灵活配置：可自定义最大引用令牌数、解析页面大小等参数

关键组件

构造函数参数

  • function_list: 自定义工具函数列表
  • llm: 语言模型实例
  • system_message: 系统提示消息
  • files: 预加载的系统文件列表
  • rag_cfg: RAG配置参数，包括：
    • max_ref_token: 最大引用令牌数（默认4000）
    • parser_page_size: 解析器页面大小（默认500）
    • rag_keygen_strategy: 关键词生成策略
    • rag_searchers: 检索器类型列表

主要方法

  • _run(): 核心运行方法，处理用户消息并检索相关文档
  • get_rag_files(): 获取待检索的文件列表

工作流程

  1. 文件提取：从用户消息中识别并提取上传的文件路径
  2. 文件过滤：验证文件类型是否支持（在PARSER_SUPPORTED_FILE_TYPES中）
  3. 关键词生成：使用LLM根据用户查询生成检索关键词（如启用）
  4. 内容检索：调用检索工具搜索相关文档片段
  5. 结果返回：将检索到的内容作为消息返回给用户

支持的文件类型

目前支持以下文件格式的解析和检索：

  • .pdf (PDF文档)
  • .docx (Word文档)
  • .pptx (PowerPoint演示文稿)
  • .txt (纯文本文件)
  • .csv (逗号分隔值文件)
  • .tsv (制表符分隔值文件)
  • .xlsx (Excel电子表格)
  • .xls (旧版Excel电子表格)
  • HTML文件

典型应用场景

  • 企业知识库：员工上传政策文档、产品手册等，供智能助手检索
  • 学术研究：研究人员上传论文，系统可提取相关信息辅助分析
  • 客户服务：客服人员上传FAQ文档，辅助解答客户问题
  • 法律咨询：律师上传法规条文，在咨询时快速检索相关内容

注意事项

如果未提供LLM模型，关键词生成功能将自动禁用（rag_keygen_strategy设为'none'）

该模块依赖于qwen_agent框架的核心组件，包括语言模型、工具系统等
